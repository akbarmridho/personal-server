# Output Report Template

Use this structure for technical outputs.

```markdown
## Technical Analysis: {SYMBOL}

### A. Header
- Symbol, date, current price, data range
- Intent: ENTRY / HOLD / EXIT / SCREENING

### B. Quick Assessment
- State: Balance or Imbalance
- Trend status and MA context
- Volume behavior summary
- Overall risk level

### C. Key Levels
- Top resistance zones
- Top support zones
- Mandatory stop-loss level

### D. Risk Assessment
- Red flags by category with severity and evidence
- Divergence status: none / unconfirmed / confirmed
- Informed-money signals if present
- Scenario risk and mitigation

### E. Action
- BUY / HOLD / SELL / WAIT / EXIT
- Rationale
- Conditions that would change recommendation

### F. Detail (if needed)
- Entry plan or hold management or exit execution detail

### G. Chart Outputs
- List generated chart files and purpose
```
