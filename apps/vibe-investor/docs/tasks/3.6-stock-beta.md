# 3.6 Stock Beta Relative to IHSG

Status: DONE
Priority: P3
Effort: Small

## Change

Compute rolling beta from existing OHLCV data (stock + IHSG daily closes).

- `beta = Cov(r_stock, r_ihsg) / Var(r_ihsg)` over rolling window
- 120-day primary, optional 60-day short-term
- Classification: defensive (<0.7), moderate (0.7-1.3), aggressive (>1.3)
- Portfolio-level: position-weighted average beta across holdings
- Integration: refine aggression multiplier — high-beta stock in weak regime gets steeper size discount

No new data source needed — `fetch-ohlcv` already has both stock and IHSG daily.
