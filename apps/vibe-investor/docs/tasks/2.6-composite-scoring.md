## Change

Replace "exit synthesis contract" and "mixed-signal arbitration" sections in main.md with composite scoring. This promotes the shadow scoring from 2.0 to the primary decision engine and removes the gate cascade.

Formula:

```
composite = 0.25 × technical + 0.15 × flow + 0.25 × narrative + 0.20 × fundamental + 0.15 × portfolio_fit
```

When a lens is skipped: use neutral default (50), or prior session's score if recent (<3 desk-checks old).

Score-to-action mapping:

| Composite | Action | Base Size |
|-----------|--------|-----------|
| 0-25 | NO TRADE | 0% |
| 26-40 | WATCHLIST | 0% |
| 41-55 | PILOT | 0.25-0.5% |
| 56-70 | STARTER | 0.5-1.5% |
| 71-85 | STANDARD | 1.5-3% |
| 86-100 | HIGH CONVICTION | 3-5% |

Hard safety rails (only binary overrides):

1. Portfolio heat >8% → block all new longs
2. Thesis invalidated from any lens → EXIT
3. Single position >30% → block adds
4. Position >5% ADTV → cap size
5. Max 3-4 active pilots simultaneously

Conflict resolution: when lenses disagree, state the conflict, assess which lens is more relevant for this symbol in this context, produce a final score with the conflict noted.

Dual mandate: "protect capital" AND "deploy capital" with equal architectural weight.
