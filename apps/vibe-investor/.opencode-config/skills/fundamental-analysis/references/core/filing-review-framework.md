# Filing Review Framework

Use this reference when `FILING_REVIEW` is active or when formal disclosures materially change a broader fundamental call.

## Mandatory Review Output

Every filing-led review must state:

- which disclosure was reviewed
- period covered
- why it is the correct primary source
- what changed
- what is filing-backed
- what remains inference

## Source Selection

| Use this document first | When the question is mainly about |
|---|---|
| Annual report / audited annual statements | business quality, accounting quality, governance, long-cycle capital allocation |
| Quarterly / interim statements | recent deterioration, inflection, current execution |
| Rights issue / placement / debt raise / material transaction disclosure | capital structure, control, funding path, dilution |
| Public expose / management presentation | management claims, capex plans, segment updates not yet fully visible in statements |

Do not use a lighter source when a stronger official source exists for the same claim.

## Review Matrix

| Document type | Mandatory checks | Escalate if |
|---|---|---|
| Annual / audited | business and segment framing, revenue and margin match, cash vs profit, debt and maturity profile, working capital, RPT balances, contingencies, auditor opinion, notes-to-accounts items, capital allocation behavior | qualified or adverse opinion, restatement, going-concern language, unexplained RPT movement, large profit-to-cash disconnect, equity growth driven mainly by repeated raises |
| Quarterly / interim | revenue and profit trajectory, margin drivers, OCF vs PAT, debt movement, working-capital movement, base-effect vs real trend | headline beat driven by one-offs, revenue not converting into cash, margin improvement unsupported by durable drivers |
| Notes to accounts | revenue recognition, segment definition changes, RPT balances, receivables aging, inventory detail, impairment, debt terms, covenants, contingencies, fair-value assumptions | notes materially weaken earnings quality, asset quality, leverage interpretation, or minority alignment |
| Auditor remarks | opinion type, key audit matters, emphasis-of-matter items, control or accounting-policy warnings | market headline is clean but auditor language is not |
| Rights issue / placement / debt raise | exact use of proceeds, refinancing vs growth, dilution, pricing quality, participant quality, balance-sheet impact | vague purpose, controller-first benefit, repeated fundraising replacing self-funded operations, debt roll without cash-generation improvement |
| Material transaction | transaction object, strategic fit, pricing sanity, leverage effect, dilution effect, cash-flow effect, related-party angle | highly levered, opaque, circular, or cosmetic deal |
| Public expose / management presentation | guidance credibility, capex claims, funding claims, explanation for margin, demand, and working-capital changes, consistency with filings | story improves faster than filings do |

## Interpretation Rules

- Notes-to-accounts can be thesis-critical even when the headline statements look clean.
- Auditor comfort is not the same as strong economics or good governance.
- Management commentary is supporting evidence, not a substitute for statements when audited numbers or formal terms are the issue.
