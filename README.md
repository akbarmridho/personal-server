# Personal Server

## Docker Network

```bash
docker network create personal_network
```

## Port List

### Filen

- `4001` as WebDAV Server.

### Adguard Home

- `53 (TCP/UDP)` DNS Server.
- `8000 (TCP)` Admin Dashboard.

### PostgreSQL

- `5433` PostgreSQL.
- `8001 (TCP)` pgweb Dashboard.

### RSS Feed

- `8002 (TCP)` mkfd.
- `8003 (TCP)` miniflux.

### Kavita

- `8004 (TCP)` Kavita.

### CouchDB for Obsidian LiveSync

- `5984` CouchDB.
  
### Beszel

- `8005` Beszel.

### Activepieces

- `8006` Activepieces.

### Lobe Chat

- `8007` Casdoor.
- `8008` Lobe.

### Immich

- `8009` Immich.
